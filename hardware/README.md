# RelapseGuard Hardware

This folder contains the available PCB design assets for the RelapseGuard wearable prototype.

## Included Files

- 3D model preview of the PCB/wearable hardware
- PCB design screenshots
- Circuit/schematic design screenshots

## Hardware Overview

RelapseGuard uses an ESP32-based wearable architecture with:

- MAX30102 for heart-rate and SpO₂ sensing
- GSR sensor for skin-conductance/stress response
- MPU6050 for movement and restlessness detection
- Coin vibration motor for private haptic alerts
- SOS button for urgent support
- Li-ion battery with charging/protection module

## Prototype Note

This is a hackathon-stage hardware design package. The current ZIP contains PCB/circuit screenshots and 3D model previews. Editable KiCad/EasyEDA source files and Gerber exports can be added in the next revision.
